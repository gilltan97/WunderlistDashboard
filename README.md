# WunderlistDashboard 

Dashboard for viewing [Wunderlist]() task completion stats.

## Installation 
```bash
git clone https://github.com/gilltan97/WunderlistDashboard.git
cd WunderlistDashboard

# Install the node modules 
npm install 

# Start the app
npm run start 
```
